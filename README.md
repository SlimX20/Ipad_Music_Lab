# 🎛️ iPad Music Lab

A simple AI music generator built and tested on iPad.

## 🔥 Features
- Generate random melodies
- Experimental AI music ideas
- Built using Replit + GitHub

## 🎧 How to use
1. Open the app (Replit link)
2. Click generate
3. Record / sample into Koala or AUM

## 🚀 Goal
Create tools for making music directly on iPad.

- MIDI export
